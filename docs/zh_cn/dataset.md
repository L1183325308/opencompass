# 数据集

FIN-EVAL数据集由4,543个选择题组成，涵盖33个不同的学术科目。这些问题收集自四个类别:会计、金融、会计认证和专业认证。Fin-eval还包括一些特别具有挑战性的题目，需要非常高级的推理能力来解决，比如中国的精算师考试。

以下为F-Eval数据集的统计数据：
