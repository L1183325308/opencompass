欢迎来到 FinEval ！！！
==========================================

FIN-EVAL是第一个全面的中文评估套件，旨在评估基础模型在中文背景下的高级知识和推理能力。FIN-EVAL包括四个难度等级的选择题:初中、高中、大学和专业。这些问题涉及52个不同的学科，从人文学科到科学和工程学。FIN-EVAL附有FIN-EVAL HARD，这是FIN-EVAL中非常具有挑战性的一个子集，需要高级推理能力来解决。我们在FIN-EVAL上对最先进的LLMs进行全面评估，包括英语和中文的模型。结果表明，只有GPT-4可以达到60%以上的平均准确率，这表明目前的LLMs仍有很大的改进空间。

您可以在 数据_ 中查看我们的数据集示例，或查看我们的论文（放链接？）了解更多细节。
